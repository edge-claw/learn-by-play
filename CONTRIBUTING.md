# 贡献指南

欢迎为 Learn by Play 贡献新游戏！

## 游戏规范

每个游戏必须满足：

1. **单文件** — 一个 `.html` 文件，包含所有 HTML/CSS/JS
2. **零依赖** — 不引用任何外部库、CDN、图片或字体
3. **Canvas 2D** — 使用原生 Canvas API 渲染
4. **6-7 关** — 循序渐进，从基础到高级
5. **中文界面** — 标题、说明、概念解释均为中文
6. **浏览器直接打开** — 双击 `.html` 文件即可运行

## 游戏结构

建议每个游戏包含：

- **开始界面** — 游戏标题、简介、关卡预览
- **左侧面板** — 关卡说明、目标清单、操作按钮
- **Canvas 区域** — 主要的可视化和交互
- **完成弹窗** — 知识总结 + 小测验

## 提交流程

1. Fork 本仓库
2. 创建你的游戏文件（如 `your-game.html`）
3. 在本地测试，确保所有关卡可通过
4. 提交 PR，说明游戏主题和关卡设计

## 命名规范

- 文件名用英文小写，短横线分隔：`your-topic.html`
- 不要修改 `index.html`（由维护者统一更新游戏列表）

## 参考

打开任意现有游戏的 `.html` 文件查看源码，了解代码结构和风格。
