## 特点

- **零依赖** — 纯 HTML + Canvas 2D，无需安装任何东西
- **单文件** — 每个游戏一个 `.html` 文件，双击打开即玩
- **循序渐进** — 每个游戏 6-7 关，从基础概念到高级应用
- **交互式学习** — 不是看视频，是亲手操作、实验、探索
